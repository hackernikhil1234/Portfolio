# Nikhil Kumar Yadav | Software Engineer Portfolio

A premium, cinematic, and futuristic developer portfolio built with React 19, Vite, and Framer Motion.

## Features
- **Cinematic Hero**: 3D portrait transitions and "Hacker" text scramble effects.
- **Premium Toolkit**: Interactive brand-specific hover states and metallic shine effects.
- **Responsive Navigation**: Smooth scroll and mobile-optimized menu.
- **Modern Tech Stack**: React, Tailwind CSS 4, Framer Motion 12, Lucide Icons.

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Start development server with `npm run dev`.
4. Build for production with `npm run build`.

## Design System
- **Colors**: Deep grayscale with cyan and brand-specific accents.
- **Typography**: Instrument Serif for headings, Inter for body text.
- **Interactivity**: Dynamic cursor-tracking shine masks and spring-based physics.
